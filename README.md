# Healthcare SQL Analytics

## Project Overview

This repository contains SQL analytics exercises and business-oriented healthcare case studies designed to demonstrate practical SQL skills used in data analytics and reporting environments.

The project focuses on healthcare operational analysis using joins, subqueries, aggregations, conditional logic, and window functions.

---

## Skills Demonstrated

- SQL Query Writing
- Data Filtering & Aggregation
- JOIN Operations
- Subqueries
- Window Functions
- CASE Statements
- Ranking Functions
- Moving Average Calculations
- Business-Oriented Data Analysis

---

## Repository Structure

- `basic-sql/` — Foundational SQL practice and filtering queries
- `joins-subqueries/` — Multi-table analysis and nested queries
- `window-functions/` — Ranking, partitions, and advanced analytics
- `screenshots/` — SQL query result previews
